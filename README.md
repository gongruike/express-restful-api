# express-restful-api
A node express restful api sample


### Endpoint

####https://api.domain.com/{product}/v1

```
get 		/contact			获取联系人列表

post 		/contact			创建联系人

get			/contact/:id		根据id获取某个联系人的信息

put 		/contact/:id		修改某个联系人的信息

delete		/contact/:id		删除某个联系人

```
